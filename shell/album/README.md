## 功能

根据目录下的相册图片（有序的数字命名），为你自动生成其对应的Json相册配置文件。

## 用法

>更新频率太高容易被腾讯云禁用

将脚本放在相册目录外面执行，文件中的路径根据自己的情况修改。

``` zsh

zsh ./auto_create_gallery.sh  >  ./gallery.json
```
