## 功能

>PING成功=存在市电，PING失败=处于断电状态（蓄电池供电中）

检测路由器是否可以PING通，若PING失败，则延迟一定时间（15分钟左右）准备关机。在执行关机前再次尝试PING检测，若PING成功，则取消关机操作。

## 目的

保护硬盘

## 用法

将脚本放在自己的服务器上每隔5分钟定时执行
